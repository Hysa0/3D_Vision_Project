Quelques commandes utiles:
git --version -> détermine la verison de Git installée

git config --global user.name 'Username' -> défini le nom de la personne
git config --global user.email 'email@...' -> défini l'email de la personne
git config --global --list -> vérifie le nom et l'email enregistré

git init -> initialise le fichier de travail git pour le projet (ajout du dossier .git)

git status -> donne le nom de l'ensemble des branches dans le projet + document non ajouté dans la sauvegarde

git add ./ -> ajout de tous les fichier en pré-sauvegarde
git commit -m 'Ce que je viens d'ajouter' -> ajout des fichier dans la sauvegarde de la branche + ajout d'un indice pour savoir ce qu'il y a 
git log -> log de la branche en cours et des dernière modifs

git checkout 'indice du log' -> permet de se déplacer dans les sauvegardes (mode spectateur)
git branch "name_new_branch" -> créer une nouvelle branche de travail
git checkout main/master -> retour dans la branche principale 
git branch -m main -> renome la branche principale en main (par défaut master)

git remote add "name" URL -> enregistre un URL sous "name"

git push -u name main -> ajoute le repertoir cloud de l'URL "name" comme lié à notre branche local main
git push --set-upstream origin "branch name" -> ajoute les fichier à la branch en ligne
git push -> same

git pull
git pull origin main -> met a jour l'intérieur d'un fichier depuis le cloud vers le local

git merge main -> quand on est dans une autre branche, merge le repertoir main dans cette branche

git reset -> a développer (--soft or --hard)
